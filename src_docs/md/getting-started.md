# Getting Started

This guide walks you through installing and configuring the project.

## Installation

=== "pip"

    ```bash
    pip install your-package
    ```

=== "uv"

    ```bash
    uv add your-package
    ```

## Configuration

Create a configuration file:

```yaml
# config.yml
setting: value
debug: false
```

## Usage

After installation, you can use the CLI:

```bash
your-command --help
```

!!! tip "Pro Tip"
    Use `--verbose` for detailed output.
